# Global Cyber Risk Intelligence Dashboard (2015-2024)

This project provides a comprehensive dashboard for analyzing global cybersecurity threats from 2015 to 2024. The dashboard visualizes trends, patterns, and insights using interactive charts and tables.

## Features
- Interactive data visualization of global cyber threats
- Year-wise and region-wise analysis
- CSV data source: `Global_Cybersecurity_Threats_2015-2024.csv`
- Built with Python and Streamlit
- Easy to deploy and customize

## Live Dashboard
Access the live dashboard here: [Global Cyber Risk Intelligence Dashboard](https://bamlh5zcoytrni9py5g4yh.streamlit.app/)

## Getting Started

### Prerequisites
- Python 3.8+
- pip

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mercydeez/Global-Cyber-Risk-Intelligence-Dashboard-2015-2024-
   cd Global-Cyber-Risk-Intelligence-Dashboard-2015-2024-
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the dashboard locally:
   ```bash
   streamlit run app.py
   ```

## Data Source
- `Global_Cybersecurity_Threats_2015-2024.csv`: Contains the raw data used for analysis and visualization.
